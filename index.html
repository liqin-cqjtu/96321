做的什么？

首页一览

享受生活

一个贴近生活的网页

主要内容分为：现代建筑、美食、自然风景、文化这四个大块。主页面用于简介四个板块的内容，对本网页主要内容进行概括，并提供到对应详细板块界面的按钮， 在每个板块简述都有了解更多按钮，该按钮实现网页的跳转，可以细致了解更多的内容。

主界面首先是导航，由动态视频为背景，该导航主要功能是可以快速到对应板块的概括。
第一个板块就是现代建筑,简介现代建筑的发展历程，中国建筑的发展时期和建筑特点。
第二个板块就是美食，简述了中华民族的三种传统美食：饺子、月饼、烤鸭。
第三个板块就是自然风景，展示了非洲自然风光和热带雨林。
第四个就是中华民族的传统文化————大熊猫、长城。

网页内容一览

首页

首页是所有网站的跳转点，主要简介了个网页的主要内容，提供网页地址和本网站的主题。

建筑

每一个城市都有可以最为象征的建筑，每一座建筑都是文化的载体,该网页介绍了各具特色的城市建筑。

美食

享受美食，享受美好生活。主要讨论了各种水果和正餐，加入水果宴会，感受自然风采

自然

第三个就是感受自然风采，先是四季一览，然后就是非洲大草原和野生动物，最后就是绿色森林，缓解视觉疲劳

文化

中国文化，主要介绍了长城、大熊猫、稻田,导航三个可以自动轮播的图片,可以实现跳转。 别看大熊猫胖乎乎、毛茸茸，就以为它只会撒娇卖萌装可爱，实则它搞起外交、拉动经济的功力可都超一流的。

浆果

果类的营养成分因果实不同而异，外果皮为一到数层薄壁细胞，中果皮与内果皮一般难以区分。 中果皮、内果皮均肉质化，含丰富浆汁。 浆果类果树种类很多，如葡萄、猕猴桃、树莓、醋栗、越橘等。

开发过程

历程一览

在最初，首先是先学习了一些web网页开发教程，在bootcss和菜鸟教程上面有详细、全面的教程。
然后根据web课所讲的内容，将其用到了主页面的开发，选择了动态视频作为首页背景，之后思考如何搭建我的网页 和网页应该具有的内容。
在比对一些bootcss实例后，我选择了现代建筑、自然、文化、美食这四个和生活相关的主题。确定主题后， 就开始了网站内容的设计，初次网页设计，我并没有什么设计灵感，不知道网页该如何布局。于是，在网上查看了大量ccs,boo的网页实例， 从上面获取了大量网页开发经验。我将我的网页分为两个部分，第一部分为简单概括，第二部分就是详细描述，就通过网页链接就完成了网页连接。
网站设计完成后，就开始学习上传网页，最后在github上提交了网页，完成了网页的制作。

上传网页方法


  • 拥有自己的github账号，在github官网上注册账号后，点击头像左侧加号。
  • 点击New repository
  • 要上传网站，名称为 “name.github.io”,name为你的用户名
  • 选择“Add a README file" ,点击确认 。
  • 之后，在Setting中选择GitHub Pages,随便选择一个主题。
  • 然后点击code,再点击"Add file",上传自己项目，最后在GitHub Pages中保存。
  • 输入网站就可以看到自己的网站了。

总结

这学期又学了一门新技术，虽然才入门，但学习是一生的事情，需要不断提高自身。
这是一门实践效果很强的学科，在棋歌网站上就可以学习html的相关知识，计算机专业的人更应该懂得如何利用网络学习。
web课程是一门非常有意思的课程，在上课的时候就可以跟着老师做出各种网页，可以实时看到网页效果。这是一门实践效
果很强的学科，在一些网站上就可以练习html的语法，甚至可以直接在网页中编写代码，非常方便于我们测试自己都程序。开始学习时,
web前面是 HTML、CSS、JavaScripe的部分。HTML是超文本标记语言，是为浏览器设计的语言，用于在Web上传输信息。
HTML的编写比较简单，只要记住基本的语法就可以。CSS是由W3C组织制定的一种用来定义样式规则(如字体、颜色和位置)的语言，
能让网页制作者有效地定制、改善网页的显示效果。JavaScript是一种轻型的脚本语言，它与HTML相结合，可以增强功能，
提高与用户之间的交互性。主要利用的是Java来进行编写。还是相对比较简单，到后面的动态网页和各种框架，就开始比
较困难，作为新手，也只能做一下静态网页了。 总体而言，学习需要持之以恒，作为程序员，需要不断学习、了解前言技术。
